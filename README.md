# General Relativity Lecture Notes

Complete lecture notes for a one-semester course on general relativity, following Wald's *General Relativity* (1984). 23 lectures covering manifolds, tensors, curvature, Einstein's field equations, gravitational waves, FLRW cosmology, and Schwarzschild geodesics, plus one supplementary note and one addendum on the distributional construction of the energy-momentum tensor.

**[Download the full PDF](GeneralRelativity.pdf)** (110 pages)

## Building from source

Requires XeLaTeX with Times LT Std, Whitney, MTPro2, and LucimaTX fonts.

```bash
./build.sh              # Full build (3 passes + bibtex)
./build.sh --draft      # Single pass (fast)
./build.sh --cmfonts    # Computer Modern fallback (no proprietary fonts)
```
